Live Server

Live server tillåter dig att enkelt, genom högerklicksmenyn, skapa en lokal webbserver som serverar din html och dina statiska filer. Live server använder dessutom en teknik som heter "Hot Reload" vilket innebär att varje gång du sparar din html eller css-fil så uppdateras sidan i browsern automatiskt. Väldigt smidigt.
