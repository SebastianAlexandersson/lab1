Laboration 1

Genvägar och tips i Visual Studio Code

CTRL+SHIFT+UP/DOWN

Det här kortkommandot gör att man kan skapa flera olika markörer på olika rader samtidigt. Detta är speciellt användbart om man har flera identiska html element och vill göra en ändring på allihopa. Med den här genvägen kan man ändra alla elementen samtidigt.

CTRL+W

Används för att stänga en flik snabbt.

CTRL+S

Kortkommando för att spara en fil. Väldigt användbart.
